Sandbox

Simple experiment for playing around with a small database.

We've created a single table, cities that has a few cities and their population. It has two fields, city and population.

Type the following to see the cities available: 

      SELECT city FROM cities

You can see both fields in the database by typing: 

      SELECT city, population FROM cities

Might be easier to type this instead: 

      SELECT * FROM cities

The * character means to display all the fields in the database row.
